# Frontend

This is the frontend of the quotes application.

## Tech Stack

-   **Framework**: React
-   **Build Tool**: Vite
-   **Language**: TypeScript

## Quick Start

### Installation

```bash
npm install
```

### Running the app

```bash
# development
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```
