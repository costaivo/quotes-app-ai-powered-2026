# Frontend

This is the frontend of the quotes application.

## Tech Stack

-   **Framework**: React
-   **Build Tool**: Vite
-   **Language**: TypeScript

## Quick Start

### Installation

```bash
pnpm install
```

### Running the app

```bash
# development
pnpm run dev

# build for production
pnpm run build

# preview production build
pnpm run preview
```
