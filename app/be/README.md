# Backend

This is the backend of the quotes application.

## Tech Stack

-   **Framework**: NestJS
-   **Language**: TypeScript
-   **Runtime**: Node.js v20.9.0

## Quick Start

### Installation

```bash
pnpm install
```

### Running the app

```bash
# development
pnpm run start:dev

# watch mode
pnpm run start:watch

# production mode
pnpm run start:prod
```

### Test

```bash
pnpm run test
```
